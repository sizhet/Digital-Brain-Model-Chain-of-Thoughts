# Contributing to DBM-COT

Thank you for your interest in DBM-COT.

DBM-COT is primarily a **structured documentation repository**
for the Digital Brain Model (DBM) structural intelligence framework.
It contains conceptual modules, algorithms, diagrams, and engineering notes.

Contributions that improve **clarity, navigation, and correctness**
are especially valuable.

---

## What you can contribute

High-value contributions include:

- Fixing typos, broken links, and numbering issues
- Improving reading paths or release navigation
- Clarifying definitions of key terms
- Suggesting diagram improvements (Mermaid / draw.io)
- Writing short review notes on specific ITEMs

You do NOT need to contribute code to help DBM-COT.

---

## Discussions vs Issues

Please use:

### GitHub Discussions
For:
- conceptual questions
- reading guidance
- algorithm interpretation
- design discussion

### GitHub Issues
For:
- broken links
- formatting problems
- incorrect references
- repository structure problems

If unsure, use Discussions.

---

## Style guidelines

Keep contributions:

- precise
- minimal
- technically grounded

When proposing changes, include:

- file path
- section name
- suggested replacement text
